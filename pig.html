Why Pig Fits This Scenario

Pig Latin shines when you need to stitch together a sequence of data preparation steps without diving into Java MapReduce. For this demonstration, I modeled the data from a chain of specialty cafés to show how Pig scripts help analysts calculate revenue, merge lookup information, and rank stores.

Each script step below shows the Pig Latin logic followed by the resulting output.

Pig Latin Script with Outputs

1. Load and Enrich Transactions

transactions = LOAD 'transactions.csv' USING PigStorage(',') AS (
    order_id:int,
    store_id:int,
    product:chararray,
    quantity:int,
    unit_price:double,
    channel:chararray,
    order_date:chararray
);

enriched = FOREACH transactions GENERATE
    order_id,
    store_id,
    product,
    quantity,
    unit_price,
    channel,
    order_date,
    (double)quantity * unit_price AS revenue;

dump enriched;

2. FILTER High-Value Orders

premium_orders = FILTER enriched BY revenue >= 40.0;
DUMP premium_orders;
(2002,1,Cold Brew Kit,2,24.0,Store,2024-03-01,48.0)
(2005,2,Espresso Beans,5,8.5,Online,2024-03-03,42.5)

3. JOIN with Store Lookup

stores = LOAD 'stores.csv' USING PigStorage(',') AS (
    store_id:int,
    store_name:chararray,
    city:chararray
);

joined = JOIN enriched BY store_id, stores BY store_id;
DUMP joined;
(2001,1,Espresso Beans,4,8.5,Online,2024-03-01,34.0,1,Brewed Awakening,Seattle)

4. GROUP and Summarize Revenue by Store

grouped = GROUP enriched BY store_id;
store_totals = FOREACH grouped GENERATE
    group AS store_id,
    SUM(enriched.revenue) AS total_revenue,
    AVG(enriched.revenue) AS avg_order_value,
    COUNT(enriched) AS order_count;
DUMP store_totals;
(1,118.0,39.333333333333336,3)
(2,60.5,30.25,2)
(3,32.0,32.0,1)

5. ORDER Stores by Revenue

ranked_stores = ORDER store_totals BY total_revenue DESC;
DUMP ranked_stores;
(1,118.0,39.333333333333336,3)
(2,60.5,30.25,2)
(3,32.0,32.0,1)

The ranked output highlights the Brewed Awakening location as the top performer, followed by Bean Scene and Café Compass.
